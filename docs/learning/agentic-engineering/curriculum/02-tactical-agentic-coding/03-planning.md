# Success is Planned: The 80-20 of Agentic Coding

> **Track**: Tactical Agentic Coding (Lesson 3)
> **Level**: Intermediate

## Summary

First came prompts, then came plans. Learn to hand off work by encoding your engineering best practices.

## Key Concepts

- **80/20 Rule**: 80% Planning, 20% Execution (by agents).
- **Encoding Practices**: Instead of writing prompts, write guides/plans that encode _how_ you ship.
- **Scaling Velocity**: Massively scaling plan and review velocity.
