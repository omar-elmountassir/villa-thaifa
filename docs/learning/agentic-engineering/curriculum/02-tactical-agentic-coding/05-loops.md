# Close The Loops: More Compute, More Confidence

> **Track**: Tactical Agentic Coding (Lesson 5)
> **Level**: Intermediate

## Summary

Transform brittle agent workflows into self-correcting systems with strategic feedback loops.

## Key Concepts

- **Closed Loop Prompts**: Agents that check their own work.
- **Self-Correction**: If validation fails, agent corrects itself.
- **Compute for Confidence**: Trading compute time for higher reliability.
