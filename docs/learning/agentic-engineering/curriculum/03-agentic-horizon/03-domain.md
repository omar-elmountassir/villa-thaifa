# Building Domain-Specific Agents

> **Track**: Agentic Horizon (Lesson 3)
> **Level**: Advanced Mastery

## Summary

Create specialized agents tailored for specific domains and use cases.

## Key Concepts

- **Business Logic**: Agents that understand _your_ business rules.
- **Constraints**: Agents that respect _your_ limitations.
- **Specialized Workflows**: Agents that follow _your_ process (e.g., Booking.com integration).
