# Multi-File Editing: STOP coding, START prompting

> **Track**: Principled AI Coding (Lesson 2)
> **Level**: Beginner

## Summary

Learn how to handle complex multi-file changes through effective prompting.

## Key Concepts

- **Shift Mindset**: Stop typing code, start commanding edits.
- **Low vs High Level**:
  - **Low-Level Prompts**: Concise, laser-focused for existing codebases.
  - **High-Level Prompts**: Quick, lazy, "fix this bug".
- **Multi-File Strategy**: How to context load multiple files effectively.
