# Spec based AI Coding with Reasoning Models

> **Track**: Principled AI Coding (Lesson 5)
> **Level**: Intermediate

## Summary

Transform specifications into working code using advanced AI reasoning.

## Key Concepts

- **Spec Prompts**: Massive prompts that define the entire feature.
- **Workflow**: Feature Requirements → Spec → Generated Code in one go.
- **Reasoning Models**: Leveraging models that "think" before they code (e.g., o1).
