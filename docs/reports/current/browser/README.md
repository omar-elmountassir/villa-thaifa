# Browser Agent Reports

> **Purpose**: Reports generated by browser-agent during web automation tasks.
> **Created**: 2026-01-20

---

## Report Types

| Type | Template | Use Case |
|------|----------|----------|
| **Session Report** | Standard | After each browser session |
| **Incident Report** | Incident | When errors/blockers occur |
| **Data Extract Report** | Standard | After scraping operations |

## Naming Convention

```
YYYY-MM-DD-browser-platform-seq.md

Examples:
2026-01-20-browser-expedia-001.md
2026-01-20-browser-hotelrunner-002.md
```

## Required Sections

1. **Status**: Success / Partial / Failed
2. **Platform**: Which OTA/platform was accessed
3. **Actions Taken**: Step-by-step operations
4. **Evidence**: Links to screenshots in `ai/output/`
5. **Data Extracted**: Any captured information
6. **Blockers**: Issues requiring human intervention
7. **Next Steps**: Recommendations
